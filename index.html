About Myself

I'm motivated , creative, analytical, and passionate about web development seeks employment as part of a dynamic software development team.

I have more than five years experience in web development responsible for developing projects in multiples fields communication, finance, stocks, push notifications, E-commerce, etc..

While I was a student, I applied my technical skills to extracurricular projects and taught myself PHP to built a greater project for tracking network signals quality and get analysis reports for the network coverage in my country, also I built an archiving system for automated documentation in an important sector in the Egyptian government.

After Graduation I continued to expand my coding knowledge and joined a software house company called Nile code where I was thrown into the deep end of PHP, RESTful APIs and delivered a responsive web app to build financial reports management system for Johnson & Johnson company, push notifications system for the TOTAL company.

After the Journey with Nilecode, I started with mitch designs to build a great project trying to collect all public transportation tickets in one hub with our product called Tazhub.

I helped a lot of startups to build their technical projects especially in backend staff (Mr print, golden, Carefer, etc ...).
To be a professional remote web developer I decided to join Microverse to upgrade my technical and soft skills to be a part of a remote and dynamic team working from any place.

As a remote Full-stack web developer am responsible for managing back-end tasks, built frontend components and i have a very good soft skills for easy communication with other remote team members.
